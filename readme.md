# Лабораторная работа №1

## Решение системы линейных алгебраических уравнений методом Гаусса

### Описание проекта

Данная программа реализует решение системы линейных алгебраических уравнений (СЛАУ) методом Гаусса.
Программа написана на языке **Kotlin** и позволяет находить решение системы уравнений, вычислять определитель матрицы, а также проверять точность полученного результата.

Размерность системы может быть от **1 до 20**.

---

# Используемый метод

В программе реализован **метод Гаусса** — прямой метод решения систем линейных уравнений.

Метод состоит из двух основных этапов:

### 1. Прямой ход

Исходная расширенная матрица системы преобразуется к **верхнетреугольному виду** с помощью последовательного исключения неизвестных.

На каждом шаге элементы под главным диагональным элементом зануляются.

### 2. Обратный ход

После получения треугольной матрицы неизвестные находятся последовательно, начиная с последнего уравнения.

---

# Возможности программы

Программа реализует следующие функции:

* ввод матрицы **с клавиатуры**
* ввод матрицы **из файла**
* вывод **расширенной матрицы**
* построение **треугольной матрицы**
* вычисление **вектора неизвестных**
* вычисление **определителя матрицы**
* вычисление **вектора невязки**
* сравнение результатов с решением библиотеки **Apache Commons Math**

---


# Формат входного файла

Файл должен иметь следующий формат:

```
n
a11 a12 ... a1n
a21 a22 ... a2n
...
an1 an2 ... ann
b1 b2 ... bn
```

где

* `n` — размер системы
* `aij` — коэффициенты матрицы
* `bi` — свободные члены

---

# Пример входных данных

```
3
2 1 -1
-3 -1 2
-2 1 2
8 -11 -3
```

# Сравнение с библиотечным решением

В программе используется библиотека **Apache Commons Math**, которая также решает систему линейных уравнений.

Полученные результаты сравниваются с библиотечным решением для проверки точности вычислений.

Если разница между решениями очень мала, значит алгоритм реализован корректно.
